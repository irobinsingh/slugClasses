What is SlugClasses?

Slug classes is an independent site meant to help UCSC students pick their classes for each new quarter. They can see what is being said about each individual class as well as each individual professor and choose which class and professor to take for their upcoming quarter.

You can help!

Being an independent site with limited funds makes it hard to update all the information about classes, so we are crowdsourcing the information. Any logged in user can update our database. And in wiki-like fashion add reviews. This keeps things unbiased as well as updated. So get out there, and do your part to help improve this non-profit site.
